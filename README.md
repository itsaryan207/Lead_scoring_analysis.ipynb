# Lead_scoring_analysis.ipynb
Built a lead scoring model to help sales teams prioritize high-potential customers. Used Python and Random Forest to analyze CRM data like website activity and page visits to predict conversion probability. Simple, effective, and data-driven.


• Customer Lead Scoring Model
• Developed a predictive system to rank potential customers based on their likelihood to buy, reducing manual lead filtering time for sales teams.
• Processed raw CRM data using Python (Pandas/Scikit-Learn) and identified that "time spent on site" was the #1 predictor of a successful sale.
• Visualized key conversion drivers to provide actionable insights for marketing strategy.
